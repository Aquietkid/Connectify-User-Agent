# Connectify — User Agent (Front-end)

This repository contains the front-end for Connectify: an open-source chat platform for individuals. It is a JavaScript-based single-page application (SPA) that provides the UI for messaging, presence, and user settings.

# Tools Used

This project is built in ReactJS and uses several other node modules, including tailwindcss, axios, redux toolkit, and others.

# How to Run?

You must have NodeJS and either npm or yarn installed. To use the project, install dependencies using

```bash
npm intall
# OR
yarn
```

Then, start a development build of the application using

```bash
npm start
# OR
yarn start
```

This runs the application and it can be accessed via your browser.

For production builds, use `npm run build` or `yarn build`. This produces less debuggable but better optimized builds ideal for production use.
